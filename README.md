\# Artificial Intelligence Coursework 🤖



This repository contains my laboratory assignments for the Artificial Intelligence course (Spring 2026). It focuses on implementing and comparing various search algorithms and metaheuristics for optimization problems.



\## 📁 Repository Structure



\* \*\*\[AI\_tema1](./AI\_tema1): Local Search - Knapsack Problem\*\*

&#x20; \* Implementation of \*\*Steepest Ascent Hill-Climbing (SAHC)\*\* for the Knapsack problem.

&#x20; \* Comparison with Random Hill-Climbing (RHC).

&#x20; \* Analysis across multiple instances with different parameter configurations.



\* \*\*\[AI\_tema2](./AI\_tema2): Metaheuristics - TSP \& Knapsack\*\*

&#x20; \* Implementation of \*\*Tabu Search (TS)\*\* and \*\*Simulated Annealing (SA)\*\*.

&#x20; \* Application on the Traveling Salesman Problem (TSP) using \*\*2-swap\*\* and \*\*2-opt\*\* movement operators.

&#x20; \* Comparative analysis between HC, SA, and TS for the Knapsack problem.

&#x20; \* Performance visualization across iterations and parameter tuning.



\* \*\*\[Docs](./Docs): Reports \& Analysis\*\*

&#x20; \* Detailed technical documentation for each assignment.

&#x20; \* Step-by-step function explanations and results tables.

&#x20; \* Performance charts and algorithm comparisons.
